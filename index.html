C basic ch5(運算符號、表達式和語句)

基本

  • () 左至右

  • + - 右至左 e.g. -6.4 +7

  • * / 左至右

  • + - 左至右 e.g. 80 + 9

  • = 右至左 e.g. s = 9

其他

  • sizeof 返回運算對象大小（上一篇有提及）

    size_t 無符號整數類型

  • % 用於求餘數（只限整數）

  • ++ ／ –

    count++ 利用count的val，後+1

    ++count 先+1，後用count的val

優先

x*y++ 等於 (x)*(y++)

因為++/-- 只能影響一個val

類型轉換

類型升級大多數不會有問題，而類型降級則會容易有bug，因為低類型未必能完整放下整個數字

當類型不匹配時，有以下三個規則

  • 無符號轉整數，額外的位會被忽略
  • 有符號轉整數，因實現而異
  • 整型轉浮點數，行為未定義

C basic ch3(數據類型)

數據類型

位、字節、和字

  • bit (位)
    • 最小的存儲空間
    • 存 0 ／ 1
  • byte(字節)
    • 1 byte == 8 bit
    • 8 byte 有256個0、1組合， 即是2^8
    • 通過2進制，可以表示0-255/一組字符char

整數 int

  • short int(16bit) & int(16/32bit)

    • -32767 - 32767

  • long int / long(32bit)

    • -2147483647 - 2147483647

  • long long int / long long (64bit)

  • C90: unsigned int / unsigned

    • 只適用非負數
    • range： 0 - 65535

  • C90: unsigned short int / unsigned short

  • C99: unsigned long long int / unsigned long long

  • 關鍵字 signed， 強調使用有符號

整數溢出

如果整數超越取值範圍，會重新從起點開始

浮點數float

  • float
  • double
  • long double

字符char

C lang 把1byte定義為char類型

注意char類型需要用單引號，因為編譯器用單引號來分別char和string

_Bool類型

true false

  • 0 代表 false
  • 1 代表 true

類型大小

sizeof 內置運算符， 需配合%zd / %u / %lu

  • int = 4 bytes
  • char = 1 bytes
  • long / long long = 8 bytes
  • double = 8 bytes
  • lond doube = 16 bytes
