C basic ch5(運算符號、表達式和語句)

基本

  • () 左至右

  • + - 右至左 e.g. -6.4 +7

  • * / 左至右

  • + - 左至右 e.g. 80 + 9

  • = 右至左 e.g. s = 9

其他

  • sizeof 返回運算對象大小（上一篇有提及）

    size_t 無符號整數類型

  • % 用於求餘數（只限整數）

  • ++ ／ –

    count++ 利用count的val，後+1

    ++count 先+1，後用count的val

優先

x*y++ 等於 (x)*(y++)

因為++/-- 只能影響一個val

類型轉換

類型升級大多數不會有問題，而類型降級則會容易有bug，因為低類型未必能完整放下整個數字

當類型不匹配時，有以下三個規則

  • 無符號轉整數，額外的位會被忽略
  • 有符號轉整數，因實現而異
  • 整型轉浮點數，行為未定義

這是一個類型降級的例子， ch = 1107 % 265等於83，即是S的ascii碼

強制轉換 (類別名)

第二行，1.5在相加前被轉換成1
