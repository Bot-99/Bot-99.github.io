運算符號優先次序

  • ( ) 左至右
  • - + ++ -- sizeof右至左
  • * / % 左至右
  • + - 左至右
  • < > <= >= 左至右
    • > 大於
    • < 小於
    • >= 大於或等於
    • <= 小於或等於
  • == != 左至右
    • == 等於
    • != 不等於
  • = 右至左
